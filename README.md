# AuditorSEC v0.1

**UA Defensive Cyber Ecosystem Audit Framework**

Фреймворк для аудиту української кібербезпекової екосистеми: експерти, медіа, технології з фокусом на регуляторну придатність, реальну обороноздатність та освітню цінність.

---

## Місія

Аудит українських кібербезпекових ресурсів, практик та інфраструктури для забезпечення прозорості та якості UA defensive cyber (громади, бізнес, КІ).

## Об'єкти аудиту

1. **Експерти/Організації:** Оцінка практичного впливу, академічної глибини та регуляторної придатності.
2. **Медіа/Канали:** Фільтрація шуму, оцінка освітньої цінності та етичного фокусу.
3. **Технології:** Аналіз зрілості (TRL) та придатності для UA-контексту (напр. Qiskit VQC+PSO).

## Структура проекту

- `schema/`: YAML-схеми метрик для різних об'єктів.
- `profiles/`: YAML-профілі конкретних об'єктів (експерти, канали, тех).
- `scoring/`: Scoring engine (Python) для розрахунку AuditorSEC score.
- `.github/workflows/`: CI/CD для авто-скорингу профілів.

## Як користуватись

1. Додати YAML-профіль у відповідну папку в `profiles/`.
2. GitHub Actions автоматично розрахує score.
3. Результати будуть доступні в артефактах workflow як `.scored.json`.
